# Сравниваем вакансии программистов

Проект собирает статистику по зарплатам разработчиков в Москве с двух популярных площадок: [HeadHunter](https://hh.ru/) (через `API Habr Career`) и [SuperJob](https://www.superjob.ru/). Для каждого языка программирования из списка загружаются все вакансии, из них извлекаются зарплаты в рублях, вычисляется среднее значение. Результат выводится в виде двух удобных таблиц.

Список языков: `Python`, `Java`, `JavaScript`, `PHP`, `C++`, `C#`, `C`, `Go`, `Ruby`, `Swift`, `TypeScript`, `Scala`, `Objective-C`, `1с`.

## Используемые технологии

- Python
- requests
- python-dotenv
- terminaltables
- [API HeadHunter](https://gist.github.com/DevmanServices/52658ee8152985e554ca09314b7f3cb0) / [API SuperJob](https://api.superjob.ru/)

## Как установить

1. **Клонируйте репозиторий** (или создайте папку с проектом).

2. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Получите секретный ключ для SuperJob**:

- Зарегистрируйтесь на [API SuperJob](https://api.superjob.ru/) как разработчик.

- Создайте приложение и получите `Secret Key`.

4. **Создайте файл `.env` в корневой папке проекта и запишите в него ключ:**

```text
SUPERJOB_SECRET_KEY=ваш_секретный_ключ
```

## Запуск
Запустите скрипт командой:

```bash
python main.py
```

В процессе выполнения вы увидите отладочные сообщения о загрузке вакансий. После завершения будут напечатаны две таблицы.
